# Selected Projects

## FX Trade Migration Tool

### Overview

Designed and developed an automated migration platform used to move FX trade positions between jurisdictions as part of a strategic business initiative.

The solution handled complex trade hierarchies, amendment rules, block trades, and near-settled positions while minimizing operational risk.

### Challenges

* Complex tree-based trade structures
* Multiple trade eligibility conditions
* Amendment restrictions
* Time-sensitive execution windows
* High operational risk if migration failed

### Outcome

* Improved migration performance by approximately 70%
* Enabled completion during active trading windows
* Supported a business initiative associated with approximately $48M USD in value realization
* Delivered with zero major operational disruption

### Technologies

Java, Sybase, Messaging Systems, Automation Frameworks

## Solace Administration Toolkit

### Overview

Built an internal administration platform that automated repetitive Solace messaging operations across non-production environments.

The objective was to reduce dependency on manual administrative processes and accelerate development workflows.

### Outcome

Manual Task: ~5 Minutes

Automated Task: ~3 Seconds

Result:

Significant reduction in manual effort and faster developer onboarding.

### Technologies

Java, Spring Boot, Solace, REST APIs, React
